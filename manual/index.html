Introduction

Cryptogy is an easy to use yet powerful and intuitive cryptography multi-tool. With features such as public key text encryption and decryption, digital signature schemes, and image encryption.

Getting Started

Cryptogy is a web application developed in Python, with Django as the framework. Thanks to advances in web development, web applications have been adopted by all types of devices worldwide. Leading to apps that work on any device. This, coupled with responsive UI and UX, means that Cryptogy can be used on phones and personal computers alike. You can access Cryptogy from our server, if it is still up. Otherwise, you can clone the repository and run it yourself. It should not take much. Once the page is loaded, you will be presented with the home screen. Here you can choose what tool to use, as well as get some information about the app.

Local Execution

After cloning the repository, you can input the following commands to run the application locally.


# Unix
virtualenv env
source env/bin/activate

# Windows
virtualenv env
.\env\Scripts\activate

# Install required libraries
pip3 install -r requirements.txt

# Start the application
python manage.py runserver # default port 8000

Landing Page

Public Key Cryptosystems

The first and second parts of the application are based on Public Key Cryptosystems.

Text Encryption/Decryption

The first part of the application consists on encrypting and decrypting text using multiple cryptosystems. All of them work in a similar way, with a parameter and clear text input to output a cipher.

RSA

Two parameters are needed for this, primes p and q. If neither value is input, they will be generated automatically. To encrypt a message input it in the Cleartext box and press the Encrypt button. To decrypt a message input it in the Ciphertext box and press the Decrypt button.

Rabin

Two parameters are needed for this, primes p and q. If neither value is input, they will be generated automatically. To encrypt a message input it in the Cleartext box and press the Encrypt button. To decrypt a message input it in the Ciphertext box and press the Decrypt button.

ElGamal

Two parameters are needed for this, a public key and a private key. If neither value is input, they will be generated automatically. To encrypt a message input it in the Cleartext box and press the Encrypt button. To decrypt a message input it in the Ciphertext box and press the Decrypt button.

Menezes-Vanstone

Two parameters are needed for this, a public key and a private key. If neither value is input, they will be generated automatically. To encrypt a message input it in the Cleartext box and press the Encrypt button. To decrypt a message input it in the Ciphertext box and press the Decrypt button.

Digital Signature Scheme

The second part of the application consists on generating a signature and verifying it against a text or file using multiple cryptosystems. All of them work in a similar way, with a parameter and clear text input to output a signature.

RSA Signature

Two parameters are needed for this, primes p and q. If neither value is input, they will be generated automatically. To generate a signature, input the message or file in the top box and press the Sign button. To validate a signature, input the message or file in the top box and the signature in the bottom box and press the Validate signature button. If the signature is valid, "Validated" will be printed out, otherwise "Not validated" will be printed out.

ElGamal Signature

Two parameters are needed for this, a public key and a private key. If neither value is input, they will be generated automatically. To generate a signature, input the message or file in the top box and press the Sign button. To validate a signature, input the message or file in the top box and the signature in the bottom box and press the Validate signature button. If the signature is valid, "Validated" will be printed out, otherwise "Not validated" will be printed out.

Image Encryption

If encrypting, a single image is needed. You need to upload an image on the Clear Image input. Press on Encrypt Image to output two images on the bottom. If decrypting, two images are needed. You need to upload both images on the bottom two inputs. Press on Decrypt Image to output an image on the top.
